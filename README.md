## About App

- Core CRUD App By Laravel ( Role based menu permission )

## Related Model
- N Menu -> N Permission
- N MenuPermission -> N Role
- 1 User 1 Role

## Default Menu
- Permission
- Menu
- Route
- Role
- User

## Install
- `composer install`
- `php artisan starmoozie:install`
- `php artisan migrate --seed`
